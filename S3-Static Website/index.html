Storage Classes

Pick the right class — save cost without sacrificing performance

🔥
S3 Standard
'Hot Data'
  • Default S3 storage class
  • Frequently accessed data
  • Multi-AZ storage for high durability
  • Very low latency & high throughput
  • Highest availability (99.99%)
  • No retrieval cost
Website images Videos Daily logs
Standard = data you touch every day.
🕒
S3 Standard-IA
'Infrequent Access'
  • Infrequently accessed data
  • Multi-AZ storage
  • Lower storage cost than Standard
  • Retrieval cost applies
  • Same durability as Standard
Monthly backups Audit reports Old logs
IA = rarely used, but must be available immediately.
📦
S3 One Zone-IA
'Single AZ'
  • Stored in ONE Availability Zone
  • Cheaper than Standard-IA
  • Lower availability
  • No multi-AZ protection
  • Data loss possible if AZ fails
  • Use only for re-creatable data
Processed logs Temp analytics Cache data
One Zone-IA = cheaper, but no second chance.
❄️
S3 Glacier
'Cold / Archive Storage'
  • Long-term archival storage
  • Extremely low storage cost
  • Retrieval is slow (minutes to hours)
  • Used for compliance & history
💎 Glacier Instant Retrieval

Archive data — millisecond access. Higher cost than Deep Archive.

💎 Glacier Flexible Retrieval

Minutes to hours retrieval. Balances cost & speed.

💎 Glacier Deep Archive

Cheapest S3 storage. Retrieval: 12–48 hours. Multi-year retention.

Bank statements Medical records Legal docs
Glacier = store it, forget it, retrieve when needed.
🧠
S3 Intelligent-Tiering
'Automatic'
  • AWS automatically moves objects between tiers
  • Based on access patterns
  • No performance impact
  • Small monitoring cost applies
  • Best when access pattern is unknown
User uploads SaaS app files Unknown behavior
Intelligent-Tiering = let AWS decide.
